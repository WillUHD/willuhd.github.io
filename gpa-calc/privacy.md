# Privacy Policy for GPA Calculator v3
> Last updated: Apr 15, 2026

Will Chen (willuhd) / Michel Guo (michelg10) ("we," "our," or "us") operate the GPA Calculator application.

### 1. Data Collection
We do **NOT** collect, store, or share **ANY** personal information, usage data, or device IDs. 

The code is FULLY open-source [here](https://github.com/michelg10/GPA-Calculator-2), and we recommend the community to actively verify our code against our privacy standards. 

GPA Calculator functions solely by downloading the remote configuration profile it needs from a hardcoded URL. 

You can inspect the URL here: [https://github.com/WillUHD/CourseResources/blob/main/Courses.gpa](https://github.com/WillUHD/CourseResources/blob/main/Courses.gpa)

### 2. Third-Party Services
The app does not use any third-party SDKs, analytics, or advertising frameworks. It sends an encryped https request to download configuration file. 

When GPA Calculator downloads the profile at startup, the server that hosts the data (GitHub proxy) may be able to see your IP address. **This is necessary for any connection to exist**, same for every other website you visit. 

We cannot control this, but we ensure we do not log anything ourselves. The app sandbox is physically disabled sending outgoing requests. 

### 3. Data Retention
Since we do not collect any data, we do not retain any data. 

### 4. Changes to This Policy
We might not ever update this privacy policy. However, if it does update, we will try our best to alert all active users.

If you have any questions about this Privacy Policy, please contact Will Chen [here](mailto:willcxd@outlook.com). I'm also available as `willcxd` on WeChat and Instagram.
